## CheatSheet

| File Manager  | key          |
| ------------- | ------------ |
| open / toggle | `<C-n>`      |
| focus         | `<leader>e`  |
| collapse      | `<leader>ec` |
| refresh       | `<leader>er` |

| Buffer           | key          |
| ---------------- | ------------ |
| delete one word  | `<C-w>`      |
| toggle word wrap | `<leader>uw` |

| Format                | key          |
| --------------------- | ------------ |
| toggle format on save | `<leader>uf` |
